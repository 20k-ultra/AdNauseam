AdNauseam est une extension de navigateur désignée dans le but d'obscurcir les données de navigation et de protéger les utilisateurs du fichage des réseaux publicitaires. Comme un bloqueur de pub, AdNauseam bloque les publicités malveillantes et cache toute publicité sur les pages que vous visitez. Mais AdNauseam permet plus, stockant chaque publicité dans son AdVault pour que vous puissiez les explorer interactivement. AdNauseam peut également être configuré pour automatiquement simuler un clic sur chaque publicité bloquée, dans le but de polluer vos données de navigations avec des données inutiles et faussées, permettant de rendre confus les réseaux de publicités et agences de surveillances qui violent votre intimité.
